# Задача 15.1: Dark mode text contrast + turn legibility

## Связь с юзер-кейсами
- UC-02: Observe a live public session
- UC-12: Mobile navigation and readable turn pairing

## Цель задачи
Повысить читаемость текста и меток в тёмной теме Telegram WebView, особенно в карточках ходов и метаданных.

## Описание изменений

### Изменения в существующих файлах

#### Файл: `apps/web/src/index.css`
- Усилить контраст `--text`, `--text-muted`, `--text-faint` для тёмной темы.
- Явно задать цвет текста для `.turn-block` и повысить читаемость `.turn-label`/`.turn-meta`.
- Ослабить/отрегулировать эффекты поверхностей (sheen/texture) для лучшей читабельности.

## Тест-кейсы

### E2E
1. **TC-E2E-15.1-01:** Тексты ходов и метаданные читаемы на мобильном тёмном фоне.

## Критерии приёмки
- [ ] Текст в карточках ходов видим в тёмной теме без необходимости увеличения.
- [ ] Метки (Researcher/Subject/время) не теряются на фоне.
- [ ] Общая контрастность соответствует требованиям (>= 4.5:1 для основного текста).
