# Задача 21.1: Отключить тяжелые анимации в TWA и стабилизировать UI

## Связь с юзер-кейсами
- UC-12: Mobile UX + UI polish

## Цель задачи
Устранить мерцание/фликер в Telegram WebApp (TWA) за счет отключения тяжелых анимаций и эффектов в TWA контексте.

## Описание изменений

### Изменения в существующих файлах
- `apps/web/src/App.tsx` — добавить класс `tma` на `<html>` при запуске внутри Telegram.
- `apps/web/src/index.css` — отключить анимации и тяжелые эффекты (beam/orb/halo/panel sweep, tab pulse, breath scan) в режиме `html.tma`.

## Детали по реализации
- В TWA отключить анимации и blur/смешивание слоев, чтобы исключить GPU flicker.
- Сохранить статичную визуальную подачу без скачков и мерцаний.

## Тест-кейсы

### End-to-end тесты
1. **TC-E2E-21.1-01:** UI стабилен в TWA без визуального мерцания (ручная проверка).

### Регрессионные тесты
- `npm run e2e` (headed, при необходимости).

## Критерии приёмки
- [ ] В TWA исчезает видимое мерцание при прокрутке/рендере.
- [ ] Дизайн остается читаемым и статичным (без резких вспышек).
