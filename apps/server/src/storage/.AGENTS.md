# Directory: apps/server/src/storage

## Purpose
Storage adapters, migrations, and retention helpers.

## Files
- db.ts — selects Postgres vs in-memory repositories.
- migrate.ts — applies SQL migrations on startup.
- retention.ts — periodic cleanup of stored records.
- redis.ts — stream buffer interface and Redis implementation.
- repositories/ — repository type re-exports.
- postgres/ — Postgres repository implementations.
- in_memory/ — in-memory repository implementations.
