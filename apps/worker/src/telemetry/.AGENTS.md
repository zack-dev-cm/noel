# Directory: apps/worker/src/telemetry

## Purpose
Derive and normalize telemetry for stream events.

## Files
- breath.ts — derive breath cadence, parse optional self-report, merge and clamp values.
- metrics.ts — compute telemetry metrics with optional breath integration.
- breath.test.ts — unit tests for cadence determinism and bounds.
- metrics.test.ts — integration tests for breath inclusion and ranges.
