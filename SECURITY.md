# Security Policy

## Reporting a Vulnerability

Please report security issues privately. Do **not** open a public issue.

Preferred method:
- Use GitHub Security Advisories for this repository.

If Security Advisories are unavailable, contact the project maintainers via a private channel (e.g., direct message or email listed in the repository).

## Disclosure Policy

- We will acknowledge receipt of your report within 72 hours.
- We will provide a remediation plan or status update within 14 days.
- Once a fix is ready, we will coordinate a public disclosure timeline.

Thank you for helping keep Noetic Mirror secure.
